# Bar Rosso 27 - Sondrio 🏁

Sito web statico ufficiale del **Bar Rosso 27** situato a Sondrio. 
Il progetto si ispira al leggendario numero 27 della Ferrari, coniugando l'estetica racing con l'atmosfera accogliente del locale (legno, crema e rosso).

## 🏎️ Concept
Il design riflette l'identità del bar:
- **Rosso Corsa:** Passione e richiamo al mito di Gilles Villeneuve.
- **Crema & Legno:** Il calore dell'accoglienza e dell'arredo interno.
- **Minimalismo:** Struttura statica veloce e leggera, ottimizzata per dispositivi mobile.

## 🛠️ Tech Stack
- **HTML5 / CSS3** (Custom Properties)
- **GitHub Pages** per l'hosting
- **Responsive Design** (Mobile-first)

## 🚀 Deployment
Il sito è distribuito automaticamente tramite GitHub Pages sulla branch `main`.
